# car-comparison-tool
A Python tool that compares specs of any two cars using real API data.

## Features
- Compare horsepower, MPG, and performance stats
- Search any make, model, and year
- Side by side comparison output

## How to run
1. Clone this repo
2. Install requirements: pip install requests
3. Run: python main.py

## Built with
- Python
- API-Ninjas Car API
